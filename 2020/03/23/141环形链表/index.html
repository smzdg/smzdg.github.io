141.环形链表

发表于 2020-03-23 分类于 LeedCode

问题描述

给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，我们使用整数pos来表示链表尾连接到链表中的位置（索引从0开始）。如果pos是-1，则在该链表中没有环。

示例

示例 1：

输入：head = [3,2,0,-4], pos = 1
输出：true
解释：链表中有一个环，其尾部连接到第二个节点。

链表：3->2->0->-4->2…

示例 2：

输入：head = [1,2], pos = 0
输出：true
解释：链表中有一个环，其尾部连接到第一个节点。

链表：1->2->1…

示例 3：

输入：head = [1], pos = -1
输出：false
解释：链表中没有环。

链表：1

来源

LeedCode Q141

法1：直接改变链表

改变值

选择一个链表中几乎不可能出现的值,每遍历一个节点就把这个节点的值改成相应的我们选择的那个几乎不可能出现的值.

改变链表方向

每遍历一个节点,就把指针的指向指到原链表中指向自己的节点.如果链表有环,那么我们必然会回到头结点.

法2：不改变链表中的值

哈希表

每到一个节点,检查在哈希表中是否已经存在,如果是,说明有环,如果没有,则就把这个节点加入到哈希表中.

快慢指针

哈希表的方法需要O(n)的空间,那么能不能只使用O(1)的空间解决这个问题呢.使用快慢指针即可.

快指针一次走两个节点,慢指正只走一个节点.进入环后两个指针必然会相遇(快指针每走过一个节点便检查一次).
