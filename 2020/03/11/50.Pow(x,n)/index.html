Pow(x,n)

发表于 2020-03-11 分类于 LeedCode

问题描述

实现 pow(x, n) ，即计算 x 的 n 次幂函数。

说明

  • -100.0 < x < 100.0
  • n 是 32 位有符号整数，其数值范围是 [−231, 231 − 1] 。

示例

来源

LeedCode Q50

问题分析

实现幂函数,我们可以采用最简单的逐个递乘的方法.但这样做毫无疑问是最低效的,我们没有充分利用到计算中间过程的结果.

一个简单而直观的改进算法是快速幂.对于Pow(x,n),显然我们可以由Pow(x,n/2)相乘或者是Pow(x,(n-1)/2)相乘再乘x得到.

空间复杂度由暴力法的O(n)降低到了O(log n)
