139.单词拆分

发表于 2020-03-24 分类于 LeedCode

问题描述

给定一个非空字符串 s 和一个包含非空单词列表的字典 wordDict，判定 s 是否可以被空格拆分为一个或多个在字典中出现的单词。

说明

  • 拆分时可以重复使用字典中的单词。
  • 你可以假设字典中没有重复的单词。

示例

例1

输入: s = "leetcode", wordDict = ["leet", "code"]
输出: true
解释: 返回 true 因为 "leetcode" 可以被拆分成 "leet code"。

例2

输入: s = "applepenapple", wordDict = ["apple", "pen"]
输出: true
解释: 返回 true 因为 "applepenapple" 可以被拆分成 "apple pen apple"。
     注意你可以重复使用字典中的单词。

例3

输入: s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
输出: false

来源

LeetCode Q139

问题分析

暴力DFS

最容易想到的解法是暴力DFS,从第一位开始与字典里的词逐一比较,直到到最后一位正好对应到字典中某个单词的末尾

但是这样的方法一般都会超时,在最坏的情况下复杂度可能高达O(n^k).

动态规划

对于字符串中的一段,能不能被拆分是一个独立的问题.就像例1中的例子,字符串中的Leet在能在字典中找到,我们只需要检查后面的部分就可以了.

f(j,i)    判断j到i在字典中是否有单词对应
